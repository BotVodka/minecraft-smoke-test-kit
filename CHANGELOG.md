# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by Keep a Changelog and uses semantic versioning where practical.

## [0.1.0] - 2026-03-17

### Added
- Initial public release of `minecraft-smoke-test-kit`
- Auto-stop orchestration script: `scripts/mc_smoke_test.py`
- Claude Code / Trellis smoke-test command template
- Cursor smoke-test command template
- Forge helper Java templates for explicit startup markers
- Chinese primary README and English alternative README
- Integration, adaptation, and command usage documentation
- Example Forge project layout

### Notes
- This release focuses on Forge integration first
- The stable contract is the explicit marker protocol plus the orchestration script
- Future NeoForge / Fabric support should be added as new helper templates
